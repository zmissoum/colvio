## Detailed Description

🔍 Colvio — The Data Explorer for Microsoft Dynamics 365 / Dataverse

Colvio is a free, open-source Chrome extension that gives D365 consultants, admins, and developers instant access to their Dataverse data — directly from the browser, with zero configuration.

No API keys. No app registration. No client secret. Just click the icon on any D365 page and start exploring.

━━━━━━━━━━━━━━━━━━━━━━━━

⚡ KEY FEATURES

▸ Data Explorer
Query any entity with a visual query builder. Smart field picker with search and type filtering. Multi-filter WHERE with AND/OR groups and parentheses. $expand for parent AND child relationships (1→N). Auto-pagination fetches all pages in background.

▸ Show All Data
Auto-detects the record open in your D365 tab. One click to inspect every field with logical name, type, and value.

▸ Data Loader
Import CSV or paste from Excel. Smart auto-mapping, lookup resolution (GUID direct + search), transforms (statecode, picklist, boolean). OData $batch for 10x faster imports. CREATE or UPSERT with primary key support.

▸ Bulk Delete
Select records with checkboxes. Confirm and delete directly from query results.

▸ Export
XLSX (native Excel), CSV, JSON. Copy to clipboard or download. All loaded records included.

▸ Metadata Browser
Browse entities, fields, and OptionSet values with codes and labels.

▸ Login History
View user login/logout timeline from D365 audit logs. Stats and CSV export.

▸ Saved Queries
Save and reload your frequent queries. Persisted across sessions.

🔒 SECURITY & PRIVACY

• Zero data collection — no analytics, no telemetry, no external servers
• All requests go directly from your browser to your D365 server
• Uses your existing Azure AD session — no credentials stored
• Input validation on all API parameters
• Open source — audit the code yourself on GitHub
